## Cores no Terminal

### Estrutura

> **\033[estiloTexto:corTexto:corFundo\033[m**

Abaixo estão os valores aceitáveis:

### Estilo do Texto

  ```python
  style = {
    "fecha":"\033[m",
    "none":"\033[0",
    "negrito":"\033[1",
    "italico":"\033[4",
    "invertido":"\033[7",
  }
  ```

### Cor do Texto

  ```python
  text = {
    "fecha":"\033[m",
    "branco":"\033[30m",
    "vermelho":"\033[31m",
    "verde":"\033[32m",
    "amarelo":"\033[33m",
    "azul":"\033[34m",
    "roxo": "\033[35m",
    "ciano": "\033[36m",
    "cinza": "\033[37m",
  }
  ```
